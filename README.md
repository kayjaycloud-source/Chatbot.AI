Python Chatbot – Beginner’s Toolkit with GenAI
📍 Overview
This project is part of the Moringa AI Capstone. It demonstrates how to use Python to build a simple chatbot, guided by generative AI prompts. The chatbot interacts with users via text and responds to greetings, farewells, and general input.

🎯 Project Goal
Learn Python basics with the help of AI prompts.

Build a minimal chatbot that runs in the terminal.

Document the learning process, common issues, and fixes.

Share a reproducible toolkit for beginners.

⚙️ System Requirements
OS: Windows/Mac/Linux

Python: Version 3.8+

Editor: VS Code, PyCharm, or any text editor

Optional: Virtual environment for project isolation

🚀 Installation & Setup
Clone the repository

bash
git clone https://github.com/yourusername/python-chatbot.git
cd python-chatbot
(Optional) Create a virtual environment

bash
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows
Run the chatbot

bash
python chatbot.py
📝 Minimal Working Example
The chatbot responds to simple inputs:

python
You: hello
Bot: Hi there! How can I help you today?

You: bye
Bot: Goodbye! Have a great day!
📓 AI Prompt Journal
Example prompt used:

“Give me a step-by-step guide to build a simple chatbot in Python.”

The AI provided setup instructions, sample code, and explained input/output handling. This accelerated learning and clarified structure.

🛠️ Common Issues & Fixes
Python not recognized → Add Python to PATH during installation.

Encoding errors in terminal → Switch terminal or enforce UTF-8 encoding.

Chatbot not responding correctly → Ensure .lower() is used to handle case sensitivity.
